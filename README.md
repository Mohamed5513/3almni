# 🎓 علمني - منصة تعليم اللغات الأجنبية

منصة تعليمية متكاملة لتعلم 11 لغة عالمية مع مساعد ذكاء اصطناعي ونظام مستويات وكويزات.

## ✨ المزايا

- 🌍 **11 لغة:** الإنجليزية، الفرنسية، الألمانية، اليابانية، الكورية، الإيطالية، البرتغالية، الروسية، البولندية، التشيكية، العربية.
- 📚 **4 مستويات** لكل لغة (مبتدئ → متقدم).
- 📝 **كويز نهاية المستوى** (يجب 80% للنجاح والانتقال).
- 🤖 **مساعد علمني الذكي** (Chat Bot يعمل بدون API).
- 🔐 **نظام تسجيل دخول آمن** مع تشفير كلمات المرور.
- 📊 **لوحة تحكم وتتبع تقدم.**
- 👨‍💼 **لوحة إدارة** للأدمن.
- 🎨 **تصميم عصري متجاوب** بالعربية RTL.

## 🚀 طريقة التشغيل المحلي

### المتطلبات
- PHP 7.4+ (يفضل 8.0+)
- MySQL 5.7+ / MariaDB
- Apache (XAMPP / Laragon / WAMP)

### خطوات التثبيت

#### 1. ضع المشروع في مجلد السيرفر
- **XAMPP:** انسخ المجلد `3almni` إلى `C:\xampp\htdocs\`
- **Laragon:** انسخه إلى `C:\laragon\www\`
- **WAMP:** انسخه إلى `C:\wamp64\www\`

#### 2. شغّل Apache و MySQL من لوحة التحكم.

#### 3. أنشئ قاعدة البيانات
افتح **phpMyAdmin** على: `http://localhost/phpmyadmin`

ثم استورد الملفات بالترتيب من مجلد `sql/`:
1. `01_schema.sql` (الجداول)
2. `02_seed_languages.sql` (اللغات والمستويات)
3. `03_seed_english.sql` (محتوى الإنجليزية الكامل)
4. `04_seed_other_languages.sql` (محتوى باقي اللغات)

> 💡 يمكنك استيراد الملفات الأربعة دفعة واحدة عن طريق استيراد كل ملف بالترتيب.

#### 4. اضبط الإعدادات
عدّل الملف `config/database.php` إذا كانت إعدادات قاعدة البيانات لديك مختلفة:

```php
define('DB_HOST', 'localhost');
define('DB_NAME', '3almni_db');
define('DB_USER', 'root');
define('DB_PASS', '');
```

وعدّل `config/config.php` إذا كان رابط الموقع مختلفاً:

```php
define('SITE_URL', 'http://localhost/3almni');
```

#### 5. افتح المتصفح
```
http://localhost/3almni
```

## 🎯 منطق المستويات والكويزات

- كل لغة لها 4 مستويات.
- المستوى 1 مفتوح دائماً.
- المستويات التالية مقفولة حتى يجتاز المستخدم المستوى السابق بـ **80% أو أكثر**.
- يمكن إعادة الكويز بدون حدود.
- النتيجة المعتمدة هي **أعلى نتيجة** يحصل عليها الطالب.

## 🤖 المساعد الذكي (AI Chat)

يعمل **بدون أي API خارجي** - يعتمد على محرك ردود ذكي مدمج يفهم:
- شرح القواعد (المضارع، الماضي، المستقبل، الشرطية، المبني للمجهول).
- نصائح تعلم اللغات وحفظ المفردات.
- ترجمة عبارات شائعة.
- إرشادات خاصة بكل لغة (الفرنسية، الألمانية، اليابانية...).

> 💡 يمكنك ربطه لاحقاً بـ OpenAI أو Gemini عن طريق تعديل `api/chat.php`.

## ➕ كيف تضيف محتوى جديد؟

### إضافة درس جديد
في phpMyAdmin، أضف صفاً في جدول `lessons`:

```sql
INSERT INTO lessons (level_id, title, intro, content, vocabulary, grammar, dialogue, exercises, order_num)
VALUES (1, 'الدرس 6: عنوانك', 'مقدمة', 'محتوى HTML', 'مفردات', 'قواعد', 'حوار', 'تمارين', 6);
```

### إضافة سؤال للكويز
```sql
INSERT INTO questions (quiz_id, text) VALUES (1, 'نص السؤال');
SET @qid = LAST_INSERT_ID();
INSERT INTO options (question_id, text, is_correct) VALUES
(@qid, 'الإجابة الأولى', 0),
(@qid, 'الإجابة الصحيحة', 1),
(@qid, 'الإجابة الثالثة', 0);
```
